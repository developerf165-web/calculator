# Кабельный Калькулятор (Cable Calculator)

Простой и удобный инструмент для расчета веса и диаметра кабельной продукции.
Аналог функционала `bystrokabel.ru`.

## Функционал

- **Поиск**: Живой поиск по базе данных кабелей (SQLite).
- **Калькулятор**: Добавление нескольких позиций в список, расчет веса для каждой длины.
- **Итог**: Подсчет общего веса всей партии.
- **Печать**: Версия для печати результатов.

## Установка и Запуск

Требуется **Python 3.x**.

1. Скачайте файлы проекта.
2. Запустите сервер командой (в консоли):
   ```bash
   python server.py
   ```
3. Откройте в браузере: `http://localhost:8001`

## Файлы

- `index.html` - Интерфейс пользователя (Frontend).
- `server.py` - Локальный веб-сервер и API (Backend).
- `cables.db` - База данных с характеристиками кабелей.

## Лицензия

Проект разработан для личного использования.
